## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/soman96/100974825_COMP3123_Assignment2.git
   cd 100974825_COMP3123_Assignment2
   ```

2. **Set up environment variables**

   ```bash
   touch .env
   ```

3. **Set the environment variables**

   ```txt
   PORT=3000
   MONGO_URI=mongodb://comp3123-mongo:27017/comp3123_db
   JWT_SECRET=secrect-hash
   ```

4. **Build & Deploy with Docker**
- docker-compose up --build -d
