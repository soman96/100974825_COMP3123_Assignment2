## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/soman96/100974825_COMP3123_Assignment1.git
   cd 100974825_COMP3123_Assignment1
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   ```bash
   touch .env
   ```

4. **Set the environment variables**

   ```txt
   PORT=3000
   MONGO_URI=your_mongo_connection_stringJWT_SECRET=your_secret_key
   ```

5. **Run the project**
   ```bash
   npm start
   ```
